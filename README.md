## 🚀 How to Use

### Clone the repository

```bash
git clone https://github.com/prakashgangurde-ux/linux-notes
cd linux-notes
```

### Run the practice script

```bash
chmod +x linux_practice.sh
./linux_practice.sh
```

The script will walk you through all 20 commands interactively — with coloured output, real examples, and pauses between sections so you can follow along at your own pace.

It cleans up after itself — no files are left behind after the script finishes.

---

## 🖥️ Requirements

- Linux (Ubuntu 20.04+ recommended) or macOS
- Python 3 installed (`python3 --version` to check)
- Bash shell (default on most Linux systems)

**Windows users:** Use WSL (Windows Subsystem for Linux) — install Ubuntu from the Microsoft Store, then run the script inside WSL.

## 📋 Commands Covered

| # | Command | What it does |
|---|---|---|
| 1 | `pwd` | Show current location |
| 2 | `ls -la` | List all files with details |
| 3 | `cd` | Change directory |
| 4 | `mkdir -p` | Create folders including nested ones |
| 5 | `clear` | Clear terminal screen |
| 6 | `touch` | Create empty file |
| 7 | `cp -r` | Copy files and folders |
| 8 | `mv` | Move or rename files |
| 9 | `rm -r` | Delete files and folders |
| 10 | `cat` | Read file contents |
| 11 | `grep -r` | Search inside files |
| 12 | `find` | Find files by name |
| 13 | `chmod` | Change file permissions |
| 14 | `sudo` | Run as administrator |
| 15 | `ps aux` | View running processes |
| 16 | `kill` | Stop a running process |
| 17 | `ping` | Test network connection |
| 18 | `curl` | Transfer data from URLs |
| 19 | `apt` | Install and manage software |
| 20 | `history` | View command history |

## ⌨️ Keyboard Shortcuts (Bonus)

| Shortcut | What it does |
|---|---|
